# SmartLog Site

Protótipo virtual do portal SmartLog para o Projeto Integrador.

## Como abrir

Dê dois cliques em `ABRIR_SMARTLOG_SITE.bat` ou abra `index.html` no navegador.

## O que está implementado

- Portal de notícias com matéria principal, busca, filtros e favoritos.
- Newsletter com validação de e-mail e registro local.
- Comunidade com publicação de pautas no navegador.
- Agenda de webinars com confirmação de interesse.
- Área de projeto com o canvas do negócio.
- Tema claro/escuro e layout responsivo.
